
箕谷SC 動画共有システム - 完全無料版 (YouTube限定公開方式) - SPA

使い方 (簡易):
1) チーム作成: 管理者は「チーム作成」欄にチーム名と招待コードを入力して作成。
   例: チーム名 = 箕谷SC, 招待コード = minotani
   作成後、招待コードを保護者に渡す。保護者はそのコードで参加する。

2) 動画の保存 (YouTubeを推奨):
   - 各保護者が自分のYouTubeアカウントで動画を「限定公開」でアップロードする。
   - 共有URL（https://youtu.be/xxxxx や https://www.youtube.com/watch?v=xxxxx）をコピーして、アプリの「動画を登録」に貼る。
   - アプリはURLとメタデータ（試合日、対戦、会場）を localStorage に保存します。

3) ハイライト/ゴール:
   - 再生中の時刻を手入力してハイライトを登録する方式です（iframeの再生時刻取得は同一ドメイン制限のため不可）。
   - ゴールシーンには Mark Goal を使ってください。ゴール一覧で抽出できます。

制限と注意:
- このアプリは完全無料を優先するため、**YouTube限定公開のURLを保存する方式**です。動画の本体は YouTubeが担当します。
- セキュリティは招待コード方式で簡易的です。本格的な認証が必要なら Firebase Auth での実装を推奨します（無料で可能）。
- localStorageにデータが保存されるため、ブラウザを変更するとデータが移行しません。チームで共有する際は「管理者がCSVでデータをエクスポート/インポートする」運用をおすすめします（将来的に同期機能を追加可能）。
